<h1>Linux - add and manage users</h1>

<h2>Description</h2>
Using Linux commands to add, manage, and delete from the system an employee of the company.
<br />

<h2>Linux commands</h2>

- <b>sudo</b> 
- <b>useradd</b>
- <b>usermod</b>
- <b>userdel</b>
- <b>chown</b>
